# Course Score Calculator

A simple web-based calculator for computing course totals using the formula:

T = 0.1·GAA + max(0.6F + 0.2·max(Q1, Q2), 0.4F + 0.2·Q1 + 0.3·Q2)

Features:
- Weekly assignment score inputs (10 weeks)
- Quiz 1 & Quiz 2 input
- Final exam input
- Auto calculation of T, breakdown, and letter grade
- “Required final for target T” calculator
- Save/load subjects locally (localStorage)
- CSV export/import
- Pure static HTML — no build step

## How to run
Just open `index.html` in any browser.
Works offline.

## Deployment
Host on:

- Vercel
- Any static hosting service
